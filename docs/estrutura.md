# Estrutura de Pastas

## Pastas ativas (produto final)
- `backend/`: API Django/DRF (produção do protótipo).
- `frontend/`: frontend final em React + Vite (`src-react/`).
- `docs/`: documentação oficial do projeto.
- `test-result/`: saídas detalhadas dos testes executados.

## Pastas legadas/arquivo histórico
- `legacy/frontend-angular/`: frontend Angular antigo (não utilizado no build atual).
- `legacy/rework-visual-reference/`: materiais de referência visual históricos.

## Regra prática
Para desenvolvimento e validação do sistema atual, usar apenas:
- `backend/`
- `frontend/`
- `docs/`
- `test-result/`
